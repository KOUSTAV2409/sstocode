### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/sstocode.git
cd sstocode

# Install dependencies
npm install
# or
yarn install
# or
pnpm install

npm run dev
# or
yarn dev
# or
pnpm dev
```
