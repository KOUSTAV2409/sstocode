### Components
- **Borders**: slate-200 (light gray)
- **Text**: slate-900 (dark), slate-600 (secondary)
- **Backgrounds**: white (main), slate-50 (sections)
- **Interactive**: slate-900 (buttons), slate-800 (hover)
- **Code**: slate-900 (background), slate-100 (text)

## Responsive Behavior

### Mobile (< 768px)
- Single column layouts
- Full-width elements
- Adjusted padding (p-6)
- Stacked navigation

### Tablet (768px - 1024px)
- 2-3 column grids
- Moderate spacing
- Optimized padding

### Desktop (> 1024px)
- Full 3-column grids
- Max-width constraints (max-w-4xl, max-w-6xl)
- Generous spacing
- Optimal readability

---

## Design Consistency

### Spacing System
- 4px grid (4, 8, 12, 16, 20, 24, 32, 40, 48, etc.)
- Consistent padding: p-6, p-12
